Fair Comparison

Each scenario implements identical features. Same CRUD operations, same auth flow, same UI. Config files (package.json, tsconfig) are counted.

Open Source

All benchmark code is on GitHub. Run the benchmarks yourself. File an issue if something's wrong.

Methodology: Lines of code counted with wc -l excluding blank lines and comments. Token counts via tiktoken (cl100k_base). Dependencies counted from package.json / requirements.txt.
